Color Used in CSS

  1. color: In CSS <color> is used to set the foreground color of an elementâ€™s text because the text is the main thing a section is showcasing.
  2. background-color: In CSS <background-color> is used to set the background color of an element, so the stuff behind the text or the space the element takes up.
  3. border-color: In CSS <border-color> is used to set the color of the border of the element, so the perimeter of the background, the edge of the space the element takes up.

CSS Color Values

  1. Named: In CSS, when using the name of a color, it will set that part of the element to that color. For example, if the CSS code is .section {color: blue;} then the color of the text in each section would be blue.
  2. Hexadecimal: In CSS, hexadecimal is how specific colors are identified using a code of six numbers and/or letters. For example, if you want the background of a section to be a standard blue in hexadecimal, you would use the hex code #0000FF like this: .section {background-color: #0000FF;}.
  3. RGB: In CSS, RGB is the combination of a certain value of red, green and blue to make other colors. For example, if you wanted the border of your sections to be a standard blue using RGB, you would use the RGB value of 0, 0, 255 like this: .section {border-color: rgb(0, 0, 255);}.
  4. HSL: In CSS, HSL stands for hue saturation and lightness, and the combination of these can make every color. The way you use the value in CSS is similar but in tis case hue is a degree on the color wheel from 0-360, and saturation and lightness are both percentages. For example if you wanted the background of a section to be a standard blue you would put this: {background-color: hsl(240, 100%, 50%);}.
